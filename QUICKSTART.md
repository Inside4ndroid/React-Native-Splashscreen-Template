# React Native Android Splash Screen

## Quick Start Commands

```bash
# Install dependencies
npm install

# Start development server
npm start

# Run on Android
npm run android

# Generate native Android files (if needed)
npx expo prebuild --platform android
```

## Splash Screen Features

This app demonstrates:
- Custom splash screen with blue (#4A90E2) background
- Automatic splash screen hiding after resource loading
- Smooth fade-in animation for main content
- Hello World UI with modern design

## First Time Setup

1. Ensure Android Studio is installed
2. Set up Android SDK (API 33+)
3. Create or start an Android emulator
4. Run: `npm install`
5. Run: `npm run android`

The splash screen will automatically display when the app launches!
